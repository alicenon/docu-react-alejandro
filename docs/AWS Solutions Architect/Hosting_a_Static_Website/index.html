Cómo Subir y Configurar un Sitio Web Estático en Amazon S3

Paso 1: Crear un Bucket en Amazon S3​

  1. Inicia sesión en la Consola de AWS.
  2. Navega a Amazon S3.
  3. Haz clic en "Create bucket" (Crear bucket).
  4. Ingresa un nombre único para tu bucket y selecciona la región.
  5. Deja las opciones predeterminadas o ajusta según sea necesario.
  6. Haz clic en "Create bucket" (Crear bucket).

Paso 2: Subir Archivos al Bucket​

  1. Dentro del bucket recién creado, haz clic en la pestaña "Upload" (Subir).
  2. Selecciona los archivos de tu sitio web o arrástralos a la interfaz.
  3. Ajusta las configuraciones de permisos y metadatos si es necesario.
  4. Haz clic en "Upload" (Subir).

Paso 3: Configurar el Bucket para Sitio Web Estático​

  1. Ve a la pestaña "Properties" (Propiedades) del bucket.
  2. Haz clic en "Static website hosting" (Hospedaje de sitios web estáticos).
  3. Selecciona "Use this bucket to host a website" (Usar este bucket para hospedar un sitio web).
  4. Ingresa el nombre del archivo de índice (por ejemplo, "index.html").
  5. Opcional: Ingresa el nombre del archivo de error personalizado si lo deseas.
  6. Guarda la configuración.

Paso 4: Configurar Acceso Público​

  1. Ve a la pestaña "Permissions" (Permisos) del bucket.
  2. Haz clic en "Bucket Policy" (Política del bucket).
  3. Utiliza la política de ejemplo a continuación para permitir el acceso público:
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Allow",
            "Principal": "*",
            "Action": [
                "s3:GetObject"
            ],
            "Resource": [
                "arn:aws:s3:::NOMBRE_DE_TU_BUCKET/*"
            ]
        }
    ]
}

Paso 5: Verificar el Sitio Web Estático​

  1. Copia el Endpoint del sitio web estático proporcionado en la configuración de hospedaje.
  2. Abre un navegador y pega el Endpoint para ver tu sitio web estático.

Con estos pasos, habrás subido y configurado con éxito un sitio web estático en Amazon S3. Puedes compartir esta documentación con otras personas o usarla como referencia para futuros despliegues.
