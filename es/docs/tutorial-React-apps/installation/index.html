Obtener las credenciales para conectarse con firebase

Creando la app de Firebase​

Accedemos a Firebase le damos a comenzar y creamos una cuenta, es posible que la session esté por defecto en tu navegador al acceder.

Creamos un proyecto en firebase le damos en agregar proyecto

Acontinuación agregamos un nombre al proyecto por ejemplo: Ecommerce React [tuNombre] Demo1. puedes darle cualquier nombre.

No hace falta habilitar Google Analitycs para este ejemplo, lo quitamos

Una vez creado el proyecto nos muestra esta ventana.

ahora tenemos que crear una app dentro el proyecto Firebase.

En configuracion general ir a Configuración del proyecto. Esto nos da una vista general de la configuración del proyecto.

Debemos desplazarnos hacia abajo y dar click en el icono </> esto nos crea la app para nuestro proyecto de react, en este caso

Podemos darle un nombre cualquiera lo unico que nos interesan seran las credenciales de la app.

Click en Registrar app

Acontinuación click en Ir a la consola las credenciales se mostrará en la configuración de tu proyecto.

Si tienes algo similar a las credenciales de abajo es que ya tienes para modificar el archivo .env, en caso contrario habría que habilitarlos.

Es posible que databaseURL no esté habilitado desde un inicio. Habrá que crearlos.

Comenzamos por Firestore database lo ubicarás desde el panel principal o bien buscando en Todos los productos >> Firestore database

Damos click en Crear base de datos la region por defecto y comenzar en modo de prueba

Cloud Firestorage o Firestore database nos servira para administrar los usuarios y lo productos de la applicación.

Ahora que lo tenemos, agregaremos una Realtime Database para sincronizar los datos en tiempo real.

Buscamos en Todos los productos >> Realtime Database

Damos click en Crear una base de datos la region que prefieras, en mi caso la dejo en de europa y también marcar el modo de prueba

Verifique que ya tenga agregado la databaseURL. En caso contrario, intente recargando la página.

Para almacenamiento se utiliza Storage

Volvemos a Todos los productos >> Storage

Le damos en Comenzar, la región te la detecta por defecto y marcarmos modo de prueba

Por ultimo necesitamos de Authentication de Firebase para Autenticar usuarios y administrarlos en nuestra aplicación.

Lo ubicamos donde siempre, en Todos los productos. Luego click en Authentication damos en Comenzar

Marcamos como proveedor nativo Correo electrónico/contraseña. Después habilitamos el primero para Correo electrónico, el segundo lo dejamos como está.

Credenciales y Proyecto React​

Apartir de aquí utilizaremos las credenciales obtenidas en Firebase.

Recuerda encontrar el archivo .env del proyecto y remplazar las credenciales que encontramos en la Descripción general >> Configuración del proyecto de la app de Firebase.

Ejecutamos el poryecto React:

yarn dev

Vamos a la app React al botón para dar de alta a un usuario administrador

Agregando objetos en Firestore Database​

Debemos localizar Firestore Database y cambiar el role del usuario que dimos de alta previamente al de ADMIN le damos en Actualizar

Ahora debemos cerrar sesión en la app React.

Volvemos a iniciar sesión en

Nos aparecerá un mensaje Welcome to admin dashboard

>>

Agregamos un producto y le damos en y listo!

Sientase libre de explorar Firebase y las apps de creaste dentro como Storage , Authentication para verificar todo lo que se ha hecho hasta ahora.
