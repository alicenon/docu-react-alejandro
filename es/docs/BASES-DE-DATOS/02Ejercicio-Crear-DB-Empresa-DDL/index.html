1. Crea una tabla Empleados con las siguientes columnas:​

CampoTipoRestricciones

  • NumEmpleado: entero, secuencia, no admite nulos y PK
  • Nombre: texto de máximo 30 caracteres y no puede estar nunca vacío
  • Edad: entero, debe ser mayor de 16 años
  • Oficina: entero
  • Titulo: texto de máximo 20 caracteres
  • FechaContrato: (fecha con hora), no puede estar vacío
  • Director: entero
  • Cuota: decimal con 2 decimales
  • Ventas: decimal con 2 decimales
CREATE TABLE EMPLEADOS(
    NumEmpleado NUMBER(6) NOT NULL PRIMARY KEY,
    Nombre VARCHAR2(30) NOT NULL,
    Edad NUMBER CONSTRAINT CHECK (Edad > 16),
    Oficina NUMBER,
    Titulo VARCHAR2(20),
    FechaContrato Date DEFAULT sysdate NOT NULL,
    Director NUMBER,
    Cuota NUMBER(10, 2),
    Ventas NUMBER(10, 2)
);

2. Crea la tabla Oficinas con los siguientes campos:​

  • Oficina: entero y PK
  • Ciudad: texto de máximo 30 caracteres
  • Region: texto de máximo 30 caracteres
  • CodDirector: contiene el código de empleado del director de la oficina, por tanto es clave foránea que hace referencia al NumEmpleado de la tabla Empleados
  • ImporteObjetivos: decimal con 2 decimales
  • ImporteVentas: decimal con 2 decimales
CREATE TABLE OFICINA(
    Oficina NUMBER(6) PRIMARY KEY,
    Ciudad VARCHAR2(30),
    Region VARCHAR2(30),
    CodDirector NUMBER(6) NOT NULL,
    ImporteObjetivos NUMBER(10, 2),
    ImporteVentas NUMBER(10, 2),
    CONSTRAINT FK_CodDirectorEmp FOREIGN KEY (CodDirector) REFERENCES EMPLEADOS(NumEmpleado)
);

3. Crea la tabla Productos con las columnas:​

  • idFabrica: texto máximo 10 caracteres PK
  • idProducto: texto máximo 20 caracteres PK
  • Descripcion: texto de máximo 30 caracteres, no puede estar vacío
  • Precio: decimal con 2 decimales
  • Existencias: entero
CREATE TABLE PRODUCTOS(
    idFabrica VARCHAR2(10),
    idProducto VARCHAR2(20),
    Descripcion VARCHAR2(30) not NULL,
    Precio NUMBER(10, 2),
    Existencias NUMBER(6),
    PRIMARY KEY (idFabrica, idProducto)
);

4. Crea la tabla Clientes con las columnas:​

  • NumCliente: entero y PK
  • Nombre: texto de máximo 30 caracteres, no puede estar vacío
  • Vendedor: clave foránea que hace referencia a la tabla Empleados
CREATE TABLE CLIENTES(
    NumCliente NUMBER PRIMARY KEY,
    Nombre VARCHAR2(30) NOT NULL,
    Vendedor NUMBER,
    CONSTRAINT FK_VENDEDOR FOREIGN KEY (Vendedor) REFERENCES EMPLEADOS(NumEmpleado)
);
